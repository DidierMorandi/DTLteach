DTLteach — Guide utilisateur

Guide d’utilisation du support pédagogique Windows 11.

HTML localWindows 11Débutants

Objectif

DTLteach aide un accompagnant ou un apprenant à parcourir les notions essentielles de Windows 11 avec des mots simples.

Ouvrir le support

  1. Ouvrir le dossier `DTLteach`.
  2. Double-cliquer sur `DTLteach.html` ou `DTLteach_Fr.html`.
  3. Conserver l’image `menu_Windows_11.png` dans le même dossier.

Parcourir les chapitres

La navigation latérale permet d’aller directement aux thèmes : menu Démarrer, bureau, fenêtres, explorateur, Internet, e-mail, USB, Wi-Fi, programmes et sécurité.

Utilisation en atelier

Progression lente

Chaque chapitre peut être lu puis reproduit sur le poste de l’apprenant.

Support de discussion

Le guide peut servir de fil conducteur pour expliquer les mots et gestes de base.

Inventaire des programmes

`programmes_presents.html` complète le guide en présentant les outils Windows 11 courants.

Adapter le contenu

Le support est un fichier HTML autonome. Un accompagnant peut ajouter un chapitre en créant une nouvelle section et un lien correspondant dans la navigation.
