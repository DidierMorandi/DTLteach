DTLteach - Reference Manual

Descriptive reference for the Windows 11 teaching material of the DTL suite.

Type: local HTML siteAudience: beginnersDependency: none

Role

DTLteach is an introductory Windows 11 learning support. It favors clarity, reassurance and gradual themes rather than technical exhaustiveness.

Covered Content

Windows fundamentals

Start menu, desktop, windows, File Explorer, display and essential settings.

Daily use

Internet, email, Wi-Fi, USB drives, useful programs and simple security rules.

Teaching inventory

programmes_presents.html describes built-in or common Windows 11 applications and tools.

Document Structure

The project is based on static HTML pages. Chapters are represented by identified sections linked to an internal navigation bar. Styles are embedded in the files to preserve standalone use.

Target Audience

The document is intended for people discovering computers, digital workshops and support contexts where simple vocabulary matters more than technical detail.

Limits

  • The material does not replace complete Microsoft documentation.
  • Screens and visible applications may vary depending on the Windows 11 edition.
  • The project contains no application logic and collects no data.
