DTLteach — Manuel de référence

Référence descriptive du support pédagogique Windows 11 de la suite DTL.

Type : site HTML localPublic : débutantsDépendance : aucune

Rôle

DTLteach est un support d’initiation à Windows 11. Il vise la clarté, la réassurance et la progression par petits thèmes plutôt que l’exhaustivité technique.

Contenu couvert

Fondamentaux Windows

Menu Démarrer, bureau, fenêtrès, explorateur de fichiers, affichage et réglages essentiels.

Usages quotidiens

Internet, e-mail, Wi-Fi, clés USB, programmes utiles et règles de sécurité simples.

Inventaire pédagogique

`programmes_présents.html` décrit les applications et outils Windows 11 intégrés ou courants.

Structure documentaire

Le projet repose sur des pages HTML statiques. Les chapitrès sont représentés par des sections identifiées, reliées à une barre de navigation interne. Les styles sont intégrés aux fichiers pour préserver l’autonomie du support.

Public cible

Le document vise les personnes découvrant l’ordinateur, les ateliérs numériques et les accompagnements ou la simplicité du vocabulaire prime sur le détail technique.

Limites

  • Le support ne remplace pas une documentation Microsoft complète.
  • Les captures et applications visibles peuvent varier selon l’édition de Windows 11.
  • Le projet ne contient pas de logique applicative ni de collecte de données.
