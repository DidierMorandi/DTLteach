DTLteach - User Guide

User guide for the Windows 11 teaching material.

Local HTMLWindows 11Beginners

Goal

DTLteach helps a facilitator or learner browse essential Windows 11 concepts using simple words.

Open the Material

  1. Open the DTLteach folder.
  2. Double-click DTLteach.html or DTLteach_Fr.html.
  3. Keep menu_Windows_11.png in the same folder.

Browse Chapters

The side navigation gives direct access to themes: Start menu, desktop, windows, Explorer, Internet, email, USB, Wi-Fi, programs and security.

Workshop Use

Slow progression

Each chapter can be read and then reproduced on the learner's computer.

Discussion support

The guide can be used as a thread to explain basic words and gestures.

Program inventory

programmes_presents.html complements the guide by presenting common Windows 11 tools.

Adapt Content

The material is a standalone HTML file. A facilitator can add a chapter by creating a new section and a matching navigation link.
